Post 2.4

Nacho con Volumen

Proceso

Ingredientes

Para prducir este pequeño aperitivo se necesitarán una masa recomendable algo fácil de moldear como lo son las tapas para pascualinas. Una lata de Atún. Una cucharada y media de Mayonesa. Para sellar la píramide se usará queso en la punta. También se recomienda usar yema de huevo para hacer los vértices pegajosos.

Primero se amasa para soltar bien la masa.

Cuando se amasa demasiado la masa de Pascualinas sesuelta mucho y es complicado moldearla, se cometió ese error y se cambió la masa por la clásica masa para quesadillas, funciona de igual manera.

Con una tijera limpia se corta la masa en una forma de estrella con cuatro puntas.

Para el relleno en este caso se está usando atún revuelto con mayonesa.

Primero se desmenuza bien el atún se pone en un plato junto a la mayonesa y se revuelve hasta que se vea como en la imágenes. Se puede agregar más mayonesa, dempende del gusto de quien los prepare o consuma.

Luego, con cuidado, se le pone el atún en medio de la masa hasta el inicio de cada triangulo y de ahí hasta arriba dejando la masa cerrar.

Debería quedar de esta manera

Finalmente se agrega un pedazo de lámina de queso para que se derrita enicma y a su vez ayude a cerrar la pirámide.

En este caso también se uso yema de huevo en los bordes de cada Triángulo para una mejor estructura y cierre de caras.
